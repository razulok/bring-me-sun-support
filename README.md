# Bring Me Sun Support

This public repository is only for user-facing bug reports and feature requests for Bring Me Sun.

- Report a problem: open a public issue using the bug-report form.
- Suggest an improvement: use the feature-request form.
- Do not post personal data, precise home addresses, access tokens, or other confidential information.
- Application source code, releases, security work, and internal planning are maintained privately and are not part of this repository.

For security-sensitive reports, do not open a public issue. Use the private contact channel shown in the app's privacy/support page.
